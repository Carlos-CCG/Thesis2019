13 The user fails to enter the intended infusion parameter value. The number entry software allows numeric wrap-around with single key presses
(Numeric wrap-around refers to the way UP and DOWN keys behave at boundary conditions for maximum and minimum values: UP changes the maximum value into the minimum; DOWN changes the minimum value into the maximum. For cursored displays, numeric wrap-around affects also the way the cursor position is changed: LEFT changes the cursor position from the left-most position to the right-most position; the RIGHT key changes the cursor position from the right-most position to the left-most position.)
